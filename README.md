# 창 밖을 보라, 창 밖을 보라, 흰 눈이 내린다!

> ❄️ 창문에 맺힌 성에를 닦으면, 시간이 흐릅니다!

**[itch.io에서 바로 플레이하세요!](https://r2turntrue.itch.io/lotw)**

간단한 시간조작 퍼즐 게임입니다. 게임잼에서 겨울을 주제로 개발되었습니다.

창의 성에가 맺힌 부분은 시간이 흐르지 않고, 성에를 닦으면 시간이 흐르는 메커니즘을 가지고 있습니다.

OpenGL을 활용해 밑단부터 엔진을 작성했습니다.
사운드는 혼자 구현하기 어려워 FMOD Studio를 사용했습니다.
